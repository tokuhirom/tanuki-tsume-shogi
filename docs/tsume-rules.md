# 詰将棋ルール整理

本アプリが採用する詰将棋ルールと、一般的な詰将棋ルールとの差異を整理する。

## 1. 一般的な詰将棋ルール

### 基本ルール

1. **攻め方は王手の連続で詰ます** — 攻め方は毎手必ず王手をかけなければならない
2. **守り方は最善を尽くす** — 最も長く持ちこたえる応手（最長抵抗）を選ぶ
3. **守り方の持ち駒は駒箱** — 盤上にも攻め方持ち駒にもない残り全駒を守り方が自由に使える
4. **唯一解** — 攻め方の正解手が各局面で1通りのみ
5. **最短手数** — N手詰はN手でのみ詰み、より短い手数では詰まない
6. **駒余りなし** — 最終局面で攻め方の持ち駒が残ってはならない
7. **無駄合い禁止** — 手数を延ばすだけで局面の本質に変化をもたらさない合駒は有効な応手に含めない

### 将棋の禁手（詰将棋でも適用）

- **打ち歩詰め禁止** — 歩を打って詰みにする手は禁手
- **二歩禁止** — 同じ筋に自分の歩が2枚存在してはならない
- **行き場のない駒の禁止** — 次に動けない位置への駒打ちは禁手（歩・香の1段目、桂の1-2段目）

## 2. 合駒と無駄合い

### 合駒（あいごま）

飛車・角行・香車などスライド系の王手に対して、玉と王手駒の間に駒を打って利きを遮断する防御手段。
守り方は駒箱から任意の駒を合駒として使える。

### 無駄合い（むだあい）

全日本詰将棋連盟の規約案による定義:

> 単に詰方に取られ手数が延びるだけで、その駒を使わなくても詰み、
> その後の詰手順の内容に変化をもたらさない合駒を無駄合といい、
> 応手に含めないものとします。

より形式的には:

- `○合、同X（王手していた駒で取る）` の2手を取り除く
- 合駒がなかった場合と同じ手順・同じ手数以下で詰むなら **無駄合い**
- 合駒した駒が後に攻め方の手順に影響を与える場合は **有効合**

### 無駄合い判定の難しさ

- 詰将棋作家100人いれば100人とも納得する定義を作ることは困難（やねうら王開発者の指摘）
- 中合い（離れた位置への合駒）が後に開き王手のルート上に影響する場合など、判定が極めて複雑
- 柿木将棋の判定アルゴリズムが参考実装として知られるが、完全な合意はない

## 3. 本アプリの現状と差異

### 実装済み

- [x] 攻め方は王手の連続
- [x] 守り方は最長抵抗（Rust/JS統一、`>=` で最後の同長手を選択）
- [x] 唯一解チェック
- [x] 最短手数チェック（N-2手で詰まないことを確認）
- [x] 駒余りなしチェック
- [x] 打ち歩詰め・二歩・行き場のない駒の禁止
- [x] 守り方の持ち駒 = 駒箱（自動計算）

### 未実装

- [ ] **無駄合い判定** — 守り方が無意味な合駒を打てるため、探索空間が爆発する
  - 特に7手詰以上の生成・検証でボトルネックになっている
  - 合駒のたびに全7駒種を試すため分岐が膨大に

### 無駄合い未実装の影響

1. **探索速度**: 守り方の合法手にすべての合駒が含まれるため、分岐数が激増
2. **パズル品質**: 無駄合いを含む応手が「最善」として選ばれうる
3. **7手詰生成困難**: 20万試行で数問しか見つからない（守り方が合駒で逃げ切るケースが多い）
